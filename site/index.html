Quickstart

Install the manager script on your OpenWrt router, then choose the small binary source from the interactive menu.

wget -O /usr/bin/tailscale-manager https://raw.githubusercontent.com/fl0w1nd/openwrt-tailscale/main/tailscale-manager.sh
chmod +x /usr/bin/tailscale-manager
tailscale-manager

The manager script and runtime shell helpers continue to come from this repository. Compressed binaries are published as GitHub Release assets.

Guides

Install on OpenWrt

Use the interactive installer to choose official or small binaries, persistent or RAM storage, and kernel or userspace networking.

Update strategy

Small binary releases are indexed below. Script updates are tracked separately so runtime fixes and new features stay visible even when the Tailscale version does not change.

Userspace mode

Prefer userspace mode when the kernel TUN module is unavailable. The manager script can switch modes and refresh runtime scripts automatically.
